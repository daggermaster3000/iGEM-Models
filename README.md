## Getting Started

The tailocins and kinetic-pinning models are implemented in Python so, you must have an environment with all the necessary libraries for it to function.



* python libraries
  ```terminal
  pip install scipy
  pip install numpy
  pip install matplotlib
  ```

### Installation

1. Clone the repo
